# Interactive Product Card and User Form with Validation and UI

## Project Title
Interactive Product Card and User Form with Validation and UI

## Description
A small, responsive web page demonstrating core HTML, CSS and JavaScript skills. The project includes a product card component and a user registration form with client-side validation, error handling, password visibility toggles and theme switching (dark/light) persisted via localStorage.

## Features Implemented
- Product card with image, title, description, price and "Add to Cart" button
- Image fallback when the product image fails to load
- User registration form with validation on blur and submit
- Inline error messages and invalid input highlighting
- Password show/hide toggles for password fields
- Prevents submission when validation fails; shows success message on success
- Dark/Light theme toggle persisted with `localStorage`

## Tools and Technologies
- HTML5, CSS3, Vanilla JavaScript (no external libraries)
