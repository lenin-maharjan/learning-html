# Inventory Management System (HTML & CSS)

## Description
A responsive front-end Inventory Management System interface built using only HTML5 and CSS3.

## Pages
- Landing Page
- Login Page
- Dashboard
- Data Management Page

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries, Animations)

## How to Run
1. Download or clone the repository
2. Open index.html in a web browser
